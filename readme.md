---- SHOPPING LIST -----

This is simple applitaction to making grocery lists. 


- Give a name of grocery list (example: "List1" "Tesco" "If you go to Lidl"...
- Add product: 
    step 1: choose category
    step 2: write name of product
    
- Delete product by name 
- Saving to .txt file
    a) you can choose .txt file or create new one
- Reading to .txt file
    a) you can choose .txt file
    b) when grocery list is readed, you can edit readed grocery list
- Sorting grocery list by product or category
